1. Start program using main.py
2. database name should be "CustDB" "SalesDB" "OrderDB"
3. reports works on the records specific schema defined by the project
   i.	cust|name|age|phone|address 	for CustDB.slc
  ii.	order|cust|date|total		for SalesDB.slc
 iii.	order|item			for OrderDB.slc
4. files should be in the root folder, else it might give error
5. for bulk load, provide the extension of the file as well
6. when a database is created, it creates .scheme for the schema and 
   .slc for storing the records
